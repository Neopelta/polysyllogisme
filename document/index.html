Installation

1. Clone the Repository

git clone <html du repository>

2. Install Node.js and npm

Windows

  1. Download Node.js:

    • Visit Node.js official website
    • Download the recommended LTS (Long Term Support) version
    • Run the installer

  2. Verify Installation:

node --version
npm --version

Linux (Ubuntu/Debian)

# Update packages
sudo apt update

# Install Node.js and npm
sudo apt install nodejs npm

# Verify installation
node --version
npm --version

3. Install Project Dependencies

# Install copyfiles development dependency
npm install copyfiles --save-dev

npm install rimraf --save-dev

Development

Compilation

# Build the project
npm run build

La commande npm run build va :

  • Nettoyer le dossier dist
  • Compiler les fichiers TypeScript
  • Copier tous les assets nécessaires (HTML, CSS, ressources, etc.)

Testing

npm run test

This command runs the tests with Jest and automatically generates a coverage/ folder containing a detailed report of test coverage. You can view this report by opening coverage/lcov-report/index.html in your browser.

Running the Web Application

Using VS Code Live Server

  1. Install the "Live Server" extension in VS Code
  2. Navigate to the project root
  3. Navigate to the dist folder
  4. Right-click on dist/html/index.html
  5. Select "Open with Live Server"

This will launch the application in your default browser using a local development server.

Note: Make sure you have run npm run build before launching the server to ensure you have the latest compiled version of the application.
