Class Polysyllogism

Représente un polysyllogisme, qui est une série de propositions reliées de manière logique. La classe valide la structure et permet de gérer les propositions associées.

checkStructure

  • checkStructure(): boolean

  • Valide la structure du polysyllogisme en vérifiant les relations entre termes.

    Returns boolean

    true si les propositions forment un polysyllogisme et les mettent dans le bon ordre, sinon false.

    • Chaque terme doit apparaître exactement deux fois, sauf les termes de la conclusion.
    • Le terme sujet et le terme prédicat de la conclusion doivent être reliés par les prémisses.
